# Optimize Marketing Expenses

This project analyzes user behavior, sales performance, and marketing efficiency to optimize marketing expenses using data-driven methods.

## Project Overview
The analysis focuses on:
- User engagement and retention
- Conversion behavior over time
- Customer acquisition costs (CAC)
- Customer lifetime value (LTV)
- Return on marketing investment (ROMI)

Based on these metrics, actionable recommendations are provided to improve marketing budget allocation.

## Data
The project uses:
- Website visit logs
- Order and revenue data
- Marketing cost data by acquisition source

## Tools & Technologies
- Python (pandas, matplotlib, seaborn)
- Jupyter Notebook
- Data visualization
- Business and marketing analytics

## Key Outcomes
- Identification of high- and low-performing acquisition channels
- Comparison of LTV, CAC, and ROMI across sources
- Strategic recommendations for reallocating marketing spend
